# r2 - radare2

radare2 - Advanced commandline hexadecimal editor, disassembler and debugger

usage:

r2 \[file]

```
command:
aaa ------------------   code analyze a or aa 
afl ------------------   function list
vvv ------------------   visualization mode
pdf@main ----------------    print disassember function 
? 0x149a -----------------    display 0x149a in another numeric base.

```
