# i3wm shortcuts

```
$mod+Enter                      新命令行
$mod+v                          定义新窗口为垂直分割
$mod+h                          定义新窗口为水平分割
$mod+f                          全屏
$mod+d                          打开菜单，非命令行开启程序，比如firefox
$mod+Shift+q                    结束窗口
$mod+num                        切换/创建工作区
$mod+Shift+num                  切换窗口到另一个工作区
$mod+Shift+r                    重启i3
$mod+Shift+Space                切换浮动窗口
$mod+Space                      在浮动和平铺窗口之间切换
$mod+a                          Focus parent

自己加的
$mod+p                          截图
$mod+Shift+CTRL+方向             调整大小                
```
